# EasyCalc24

EasyCalc24 is a simple and fast web application that provides useful online calculators and conversion tools.
The goal of this project is to make everyday calculations quick and accessible for everyone.

## 🌐 Live Website

https://easycalc24.netlify.app/

## 🚀 Features

* 📊 CGPA Calculator
* 📐 Geometry Formula Calculator
* 🔄 Unit Converter
* 🧮 Basic Calculation Tools
* ⚡ Fast and lightweight interface
* 📱 Mobile-friendly design

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Netlify** (Deployment)
* **GitHub** (Version Control)

## 💡 Purpose of the Project

EasyCalc24 was built to provide free online tools that help users perform quick calculations without installing any software.
